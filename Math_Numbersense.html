In number sense we have learnt many types of numbers :

  • Natural numbers: Range from 1 to infinity
  • Whole numbers: Range from 0 to infinity
  • Real numbers: Range from -Infinity to +Infinty including decimals
  • Integers: Are whole numbers that range from -Infinity to +Infinty
  • Rational numbers: Are numbers that are expressed in fractional value
  • Fractions: Are same as Rational numbers
  • Ratios: Are numbers that hold value with respect to another value
  • Decimals: Are a part of real numbers with decimal value

    • We also learnt different Number systems :

  • Indian Number System
  • International Number System
  • Roman Number System

    • Using Number system for :

  • Tally Marks : Tally Marks were available from early ages
  • Expanded form : Number is expanded in numerical format or Number name format
  • Converting one number system to another : Converting number systems from Indian to International/Roman Number systems and vice versa
